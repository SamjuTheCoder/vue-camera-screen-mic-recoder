# App Recorder
### this project allow users to record camera, screen and audio using the inbuild browser MediaRecorder Api.
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
